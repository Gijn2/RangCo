**IP 기반 성인 관람 공연/전시**
: 19금 테마를 예술적인 공연(뮤지컬, 연극)이나 미디어 아트 전시 등 제한된 형태의 문화 콘텐츠로 소화하여 법적인 문제를 최소화하고 문화적 가치를 높이는 방안을 검토해야 합니다.

**크루즈 선 내 시설 활용**
: 크루즈 선박 내에서는 공해(公海)를 운항할 경우 국내 법규의 일부 제약을 벗어나 성인 전용 쇼, 카지노(승객 국적에 따라 다름) 등 성인 엔터테인먼트 시설을 운영하는 것이 비교적 용이하므로, 크루즈 선박 자체를 성인 엔터테인먼트 공간으로 활용하는 것이 현실적입니다.
